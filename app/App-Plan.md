# Stack for this project

## UI: Flutter
- Good library support. Maps, charts, sensors.
- Bluetooth support. PWA is limited on iOS.
- Stable on iOS and Android.

## Backend: Go (Gin)
- Strict routing and middleware.
- Clear separation of concerns.
- Fast and predictable performance.
- Easy to justify for backend engineering.

## Database: PostgreSQL
- Relational and reliable.
- Strong data integrity.
- Good fit for workouts, users, and social data.

## Authentication: Supabase Auth
- No custom auth logic.
- JWT-based authentication.
- Easy integration with Go.

## Realtime: WebSockets (Go)
- Live activity feed.
- Notifications.
- Online status.

## Storage: Supabase Storage
- User avatars.
- Workout images.
- Simple access control.

## Hosting
- Go API: Fly.io or Render.
- Database: Supabase Postgres.
